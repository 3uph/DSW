Vistas con facetas


Las facetas o "navegación facetada" o "búsqueda facetada" son una herramienta de
navegación que permite filtrar unos resultados en función de diferentes aspectos o "facetas".
Se puede leer más aquí: https://es.wikipedia.org/wiki/Búsqueda_por_facetas

http://dsw-base-d8-exemple.dsicv.upv.es/drupal2/web/receptes
Las facetas son un sistema aparentemente sencillo, para el usuario, pero técnicamente son
complejos. Es necesaria una gestión especial de las bases de datos que no todos los sistemas
tienen.

Para la instalación en Drupal necesitamos dos módulos: Search API y Facets. El módulo Search
API permite una gestión avanzada de la base de datos para búsquedas, y el módulo Facets
proporciona el sistema de facetas.

En primer lugar, utilizaremos composer para instalar los módulos en el directorio de sitio
Drupal:

# composer require ' drupal/search_api'

# composer require 'drupal/facets'

                  Bloques personalizados
Drupal 9 proporciona un sistema sofisticado de bloques personalizados que permite generar
bloques de contenido que están pensados para colocar en las zonas secundarias de la página:
pies, cabecera, columnas laterales, etc.. Así, los contenidos (y sus tipos de contenidos" están
pensados para suministrar el contenido principal de la página, y los bloques personalizados (y
sus tipos de bloques personalizados) sirven para proporcionar los contenidos secundarios .

Uso de los bloques personalizados


Los bloques personalizados se controlan desde Estructura > Disposición de los bloques >
Biblioteca de bloques personalizados. En esta página hay dos pestañas: Bloques y Tipos de
bloques. En Bloques se encuentran una lista de los bloques que se han creado, y en Tipos de
bloques la definición de los tipos, que funciona de forma muy parecida a la definición de tipo
de contenidos (con la posibilidad de añadir campos y configurar la presentación de Front end y
la del formulario de edición). Por defecto, Drupal 9 proporciona un tipo de bloque denominado
bloque básico que tiene un título y un cuerpo de texto.

Como ejercicio, crea tres bloques personalizados para colocar en las regiones: "Featured
bottom first", "Featured bottom second" y "Featured bottom third". Selecciona que no se
muestre el título delo blqoque, que tenga contenidos maquetados de forma diferente, y que
se muestren únicamente en la página principal del sitio web (esto se encuentra en la
configuración de cada uno de los bloques).
Gestión de los banners de publicidad usando bloques personalizados


Para gestionar la publicidad de la página, se creará un tipo de bloque personalizado
denominado "Banner" que contenga los siguientes campos:

       una foto
       un campo de enlace (una URL)
       un campo booleano para indicar si es un anuncio destacado o no.




Añade como mínimo 5 o 6 banners para poder hacer pruebas. Te sugerimos que pongas fotos
cuadradas dado que es el formato que desearemos más adelante.
